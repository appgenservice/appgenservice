App Gen Service

What is App Gen Service ?

A software platform to deploy your web application without much effort

How does it work?

  • Provide Git Repo URL
    Provide us your application(java, springboot, nodejs, reactjs..) gitrepo url.
  • Admin Panel
    We will first build, deploy your application and provide access to application admin panel to manage it.
  • Data persistence
    A new MYSQL Database created for your application to persist data.
  • Jenkins (CI/CD)
    Jenkins CI / CD pipeline for you to build and deploy upon on code checkin.
  • Docker
    We then Dockerize your application, deploy it, provide access and make it ready to handle high traffic
  • Swagger UI
    Provide you Swagger UI to play with your application REST endpoints
  • NoSQL
    Don't worry if you have NoSQL requirement, we got MongoDB available for you
  • Apache Kafka
    We are actively working to make Kafka Stream processing available for you.
  • RabbitMQ
    Soon we will be ready to serve you with RabbitMQ pub-sub.
  • Redis
    We understand you would love to store data in-memory for faster access, give us sometime to set it up for you :)
  • Kubernetes
    Coming soon..
  • istio
    Coming soon..
  • ELK observability
    Coming soon..

    Awesome, your contributions are welcome! write to us: contact@appgenservice.com
